# Mini-Trello
Backend-API, повторяющий базовый функционал Trello

## Stack
- Python
- FastAPI
- PostgreSQL
- SQLModel
- Alembic
- Docker
- Docker-compose
- Nginx

## Установка
1. Установите Docker
2. Клонируйте репозиторий:<br>
```git clone https://github.com/akimov246/Mini-Trello.git```

## Конфигурация
Переменные окружения должны храниться в файле `.env`.<br>
Замените значения в файле `.env.example` на свои и переименуйте его в `.env`.

## Запуск
1. Перейдите в папку с проектом
2. В папке nginx запустите скрипт `generate_cert_and_priv_key.sh`. 
Данный скрипт генерирует самоподписанный сертификат и закрытый ключ.
Они необходимы для работоспособности приложения по HTTPS.
3. Выполните команду:<br>
    ```docker-compose up -d --build```

## Использование
Взаимодействие с API осуществляется через интерактивную документацию Swagger UI.
- Перейдите в браузере по адресу <a href="https://localhost/docs">https://localhost/docs</a>.
- Авторизуйтесь (зеленая кнопка Authorize)
- На эндпоинтах жмите кнопку «Try it out», чтобы отправить нужный запрос

## Возможности
- Регистрация и аутентификация (JWT/OAuth2)
- Управление досками, колонками и задачами (CRUD)
- Привязка пользователей к задачам
